# flutter_hive_basic_usage


To use hive, 

pubspec.yaml  => 


dependencies:

  hive: ^2.2.3
  hive_flutter: ^1.1.0

dev_dependencies:
  
  hive_generator: ^1.1.3
  build_runner: ^2.2.0

for more info: Hive homepage; https://docs.hivedb.dev/#/
